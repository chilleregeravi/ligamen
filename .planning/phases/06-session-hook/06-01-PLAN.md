<task type="auto">
  <name>Task 2: Create session-start.sh hook script</name>
  <files>scripts/session-start.sh</files>
  <action>
Create `scripts/session-start.sh` following the verified pattern from the research. The script must:

1. **Shebang and safety:** `#!/usr/bin/env bash` with `set -euo pipefail`

2. **Disable guard (SSTH-04):** First executable line checks `ALLCLEAR_DISABLE_SESSION_START`. If set to any non-empty value, exit 0 immediately with no output.
   ```bash
   [[ -n "${ALLCLEAR_DISABLE_SESSION_START:-}" ]] && exit 0
   ```

3. **Read stdin (SSTH-03):** Capture full stdin JSON into `INPUT` variable using `INPUT=$(cat)`. Extract fields using jq per PLGN-07 pattern:
   - `SESSION_ID` from `.session_id`
   - `CWD` from `.cwd` (fallback to `$PWD` if empty)
   - `EVENT` from `.hook_event_name`
   If jq is not available, exit 0 silently (never block).

4. **Deduplication (SSTH-05):** If SESSION_ID is non-empty, check for `/tmp/allclear_session_${SESSION_ID}.initialized`. If the file exists, exit 0 silently (already ran). If absent, create it with `touch`.

5. **Project detection (SSTH-02):** Source `lib/detect.sh` using `${CLAUDE_PLUGIN_ROOT}/lib/detect.sh`. Call `detect_project_type "$CWD"` and capture the result. If CLAUDE_PLUGIN_ROOT is unset, try the relative path `$(dirname "$0")/../lib/detect.sh`. If detect.sh is not found, set PROJECT_TYPES to empty string (graceful degradation).

6. **Build context message (SSTH-02):** Construct a context string:
   - If project types detected: "AllClear active. Detected: {types}."
   - If no types detected: "AllClear active."
   - Append: " Commands: /allclear (quality gate), /allclear impact (cross-repo), /allclear drift, /allclear pulse, /allclear deploy."

7. **Output JSON (SSTH-01):** Emit the hookSpecificOutput.additionalContext JSON to stdout. Use `$EVENT` (read from stdin) as the hookEventName — do NOT hardcode. Use `jq -Rs .` to safely escape the context string. The exact format:
   ```json
   {"hookSpecificOutput":{"hookEventName":"<EVENT>","additionalContext":"<escaped context>"}}
   ```
   Build this with printf and jq for safe escaping. If jq is unavailable for escaping, use a simple printf with manual quote escaping as fallback.

8. **Exit:** Always `exit 0`. Wrap the entire body in a function or use a trap so that any unexpected error still exits 0. Use:
   ```bash
   trap 'exit 0' ERR
   ```
   at the top (after set flags).

Make the script executable: `chmod +x scripts/session-start.sh`.

IMPORTANT constraints from research:
- Never run formatters, linters, or any tool commands (SSTH-03)
- Never emit debug output to stdout — stderr only for debug (PLGN-08)
- Always exit 0 regardless of errors (non-blocking)
- Use dynamic hookEventName from stdin, not hardcoded (Pitfall 3)
