   ```json
     "name": "allclear",
   ```

   ```json
   {
     "description": "AllClear plugin hooks — auto-format, auto-lint, file guard, session start",
     "hooks": {
       "PostToolUse": [
         {
           "matcher": "Write|Edit|MultiEdit",
           "hooks": [
             {
               "type": "command",
               "command": "${CLAUDE_PLUGIN_ROOT}/scripts/format.sh",
               "timeout": 10
             },
             {
               "type": "command",
               "command": "${CLAUDE_PLUGIN_ROOT}/scripts/lint.sh",
               "timeout": 10
             }
           ]
         }
       ],
       "PreToolUse": [
         {
           "matcher": "Write|Edit|MultiEdit",
           "hooks": [
             {
               "type": "command",
               "command": "${CLAUDE_PLUGIN_ROOT}/scripts/file-guard.sh",
               "timeout": 10
             }
           ]
         }
       ],
       "SessionStart": [
         {
           "matcher": "startup|clear|compact",
           "hooks": [
             {
               "type": "command",
               "command": "${CLAUDE_PLUGIN_ROOT}/scripts/session-start.sh",
               "timeout": 15
             }
           ]
         }
       ]
     }
   }
   ```

   - `skills/cross-impact/SKILL.md`: name `cross-impact`, description "This skill should be used when the user invokes /allclear impact, asks about cross-repo references, or wants to check if a change breaks other repos."

   - `skills/pulse/SKILL.md`: name `pulse`, description "This skill should be used when the user invokes /allclear pulse, asks about service health, or wants to check running service versions."
   - `skills/deploy-verify/SKILL.md`: name `deploy-verify`, description "This skill should be used when the user invokes /allclear deploy, asks about deployment state, or wants to compare expected vs actual cluster configuration."

   Each SKILL.md body: `# AllClear {Skill Name}\n\n{Skill name} skill — implementation in progress.`

5. **Placeholder hook scripts** — Each script gets:
   - Shebang: `#!/usr/bin/env bash`
   - Comment header: `# AllClear — {name}.sh`
   - Comment: `# Placeholder: {purpose} implementation in Phase {N}`
   - Comment: `# {EventType}: {matcher}`
   - Comment: `# Non-blocking: always exits 0` (for PostToolUse/SessionStart scripts)
   - For file-guard.sh: `# PreToolUse: Write|Edit|MultiEdit` and `# Blocking: exits 2 to deny, 0 to allow`
   - Body: `exit 0`
   - Run `chmod +x` on all four scripts after creation

6. **Placeholder lib files** — `lib/detect.sh` and `lib/siblings.sh`:

   - Empty function stubs (no body)
   - Run `chmod +x` on both

