Users who type these get no response — their input is ignored and findings are returned unmodified with a stderr warning that is invisible to Claude.

Purpose: Users should be able to express agreement naturally. Ambiguous input should be surfaced as a re-prompt rather than silently ignored.

Output: confirmation.js exports AFFIRMATIVE_SYNONYMS set and NEEDS_REPROMPT sentinel. applyEdits() normalizes synonyms to "confirm" and returns NEEDS_REPROMPT for unrecognized instructions instead of the original findings array.
