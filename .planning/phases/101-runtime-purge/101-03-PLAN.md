<objective>
Rename the runtime-deps npm package from `@ligamen/runtime-deps` to `@arcanon/runtime-deps`, and strip the single remaining `Ligamen` reference from `scripts/install-deps.sh`.

**Purpose:** The package name is the last piece of package identity still encoding the old product name. Since `install-deps.sh` uses full-file diff against a sentinel (not version-based idempotency) per the v0.1.1 precedent documented in CONTEXT.md, the rename will invalidate the sentinel exactly once — users will re-download deps on next session start under the new package name. Acceptable.

**Output:**
1. `runtime-deps.json` `name` field is `@arcanon/runtime-deps`.
2. `scripts/install-deps.sh` line 2 header comment says Arcanon, not Ligamen.
3. Fresh `npm install --prefix plugins/arcanon --omit=dev --no-fund --no-audit --package-lock=false` produces `node_modules/` with no `@ligamen/runtime-deps` traces.
</objective>
