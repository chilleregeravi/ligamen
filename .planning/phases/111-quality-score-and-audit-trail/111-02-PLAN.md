<objective>
Wire the quality-score computation into `endScan()`, expose it via QueryEngine
methods and a new HTTP endpoint, and surface the score at the end of
`/arcanon:map` output and in `/arcanon:status` output.

Purpose: Users see scan quality without inspecting the database. Score is
computed from `connections.confidence` aggregates and persisted to
`scan_versions.quality_score` on every successful scan-bracket close.

Output: One QueryEngine method extended (`endScan`) + two new methods
(`getQualityScore`, `getScanQualityBreakdown`); one new HTTP endpoint; two
slash-command surface tweaks; comprehensive node tests.
</objective>

    **(c) New methods on QueryEngine class** (insert near the other read methods, after `getRepoByPath`):

    ```
    /**
     * Returns the persisted quality_score for a scan version, or null if absent
     * (column missing, scan_version not found, or score never written).
     * @param {number} scanVersionId
     * @returns {number | null}
     */
    getQualityScore(scanVersionId) {
      if (!this._stmtSelectQualityScore) return null;
      try {
        const row = this._stmtSelectQualityScore.get(scanVersionId);
        return row?.quality_score ?? null;
      } catch { return null; }
    }

    /**
     * Returns the quality breakdown for a scan version.
     * Shape: { scan_version_id, total, high, low, null_count,
     *         prose_evidence_warnings, service_count, quality_score, completed_at }
     *
     * prose_evidence_warnings: TODO(post-v0.1.3) — backed by a future
     *   scan_versions column once TRUST-02 stderr counts are persisted.
     *   Returns 0 today.
     *
     * Returns null if scan_version_id not found or table absent.
     * @param {number} scanVersionId
     */
    getScanQualityBreakdown(scanVersionId) {
      if (!this._stmtSelectQualityBreakdown) return null;
      try {
        const breakdown = this._stmtSelectQualityBreakdown.get(scanVersionId);
        if (!breakdown) return null;
        const sv = this._db.prepare(
          "SELECT id, completed_at, quality_score, repo_id FROM scan_versions WHERE id = ?"
        ).get(scanVersionId);
        if (!sv) return null;
        const serviceCount = this._db.prepare(
          "SELECT COUNT(*) AS n FROM services WHERE scan_version_id = ?"
        ).get(scanVersionId)?.n ?? 0;
        return {
          scan_version_id: scanVersionId,
          total: breakdown.total ?? 0,
          high: breakdown.high ?? 0,
          low: breakdown.low ?? 0,
          null_count: breakdown.null_count ?? 0,
          prose_evidence_warnings: 0, // TODO(post-v0.1.3): persist counter
          service_count: serviceCount,
          quality_score: sv.quality_score,
          completed_at: sv.completed_at,
        };
      } catch { return null; }
    }
    ```
