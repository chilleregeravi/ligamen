<objective>
Ship `/arcanon:list` — a read-only command that prints a five-line overview of the current Arcanon-scanned project: repo count, services partitioned by type, connection count partitioned by confidence, external actor count, and hub sync status. Composes from existing endpoints; no schema changes, no DB writes.

Purpose: NAV-01. Operators currently have no single command that surfaces "what's in this project map" without launching the graph UI. `/arcanon:list` fills that gap and is the foundation for the navigability track of v0.1.4.

Output: New markdown command file, new `cmdList` handler in `worker/cli/hub.js` registered in HANDLERS, new shared `_arcanon_is_project_dir()` helper in `lib/worker-client.sh` (also consumed by Plan 114-03), and a bats E2E test cloned from `verify.bats` placed at the repo-root `tests/` directory.
</objective>
