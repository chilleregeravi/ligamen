    1. Add imports at the top of detail-panel.js:
       - `import { render } from "./renderer.js";`
       - Keep existing `import { state } from "./state.js";`

    2. Add `selectAndPanToNode(nodeId)` as a module-scoped function (not exported — internal):
       - Look up `const pos = state.positions[nodeId]`; if undefined, return immediately (no-op)
       - Look up `const targetNode = state.graphData.nodes.find(n => n.id === nodeId)`; if not found, return (no-op)
       - Compute pan: read the canvas element dimensions from `document.getElementById('graph-canvas')`.
         Use CSS dimensions (`canvas.clientWidth`, `canvas.clientHeight`) as the viewport size.
         Center formula (mirrors the fit-btn logic in graph.js):
         ```
         const cssW = canvas.clientWidth;
         const cssH = canvas.clientHeight;
         state.transform.x = cssW / 2 - pos.x * state.transform.scale;
         state.transform.y = cssH / 2 - pos.y * state.transform.scale;
         ```
         (Do NOT change state.transform.scale — preserve current zoom level)
       - Set `state.selectedNodeId = nodeId`
       - Call `showDetailPanel(targetNode)`
       - Call `render()`

    3. Wire click delegation. Add a single `attachConnTargetListeners()` function:
       ```javascript
       function attachConnTargetListeners() {
         const content = document.getElementById("detail-content");
         if (!content) return;
         // Remove old listener by using a named handler stored at module scope
         content.removeEventListener("click", _onConnTargetClick);
         content.addEventListener("click", _onConnTargetClick);
       }
       ```
       And the module-scoped handler:
       ```javascript
       function _onConnTargetClick(e) {
         const span = e.target.closest("[data-node-id]");
         if (!span) return;
         const nodeId = Number(span.dataset.nodeId);
         if (!nodeId && nodeId !== 0) return;
         selectAndPanToNode(nodeId);
       }
       ```
       Call `attachConnTargetListeners()` at the END of `showDetailPanel()` (after `content.innerHTML = html`).
       The removeEventListener+addEventListener pattern is idempotent and safe across repeated panel opens.

    4. Add `data-node-id` attributes to every `.conn-target` span in the renderers.
       Identify the node ID at render time using the edge's source_service_id / target_service_id.

       In `renderServiceConnections(outgoing, incoming, nameById)`:
       - The function currently receives `nameById` but not the raw edges with IDs.
       - Change signature to accept edges directly: keep `outgoing` and `incoming` as edge arrays (they already are).
       - For outgoing: target id = `e.target_service_id` → `<span class="conn-target" data-node-id="${e.target_service_id}">...`
       - For incoming: source id = `e.source_service_id` → `<span class="conn-target" data-node-id="${e.source_service_id}">...`

       In `renderLibraryConnections(node, outgoing, incoming, nameById)`:
       - "Used by" incoming loop: source id = `e.source_service_id` → `<span class="conn-target" data-node-id="${e.source_service_id}">...`

       In `renderInfraConnections(node, outgoing, nameById)`:
       - "Wires" outgoing loop: target id = `e.target_service_id` → `<span class="conn-target" data-node-id="${e.target_service_id}">...`

       In `renderActorDetail(node)` — "Connected Services" section:
       - Each `cs` has `cs.service_id` (from actor.connected_services shape in graph.js).
       - Add `data-node-id="${cs.service_id}"` if `cs.service_id` is defined; otherwise omit.

    5. Style: add `cursor: pointer` to `.conn-target` via inline style or rely on CSS.
       Inline is safest: update the span template to include `style="cursor:pointer"`.
