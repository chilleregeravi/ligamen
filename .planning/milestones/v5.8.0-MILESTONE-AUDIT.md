### Documented limitations (intentional, deferred):
- **TYPE-03 C# `partial class`** — fragments across files treated as separate types. Documented in code comment in `drift-types.sh`. Cross-file merging deferred to v5.9+.
- **Hub feature flag default off** — `hub.beta_features.library_deps` defaults `false` until THE-1018 (hub-side resolver) ships. Plugin emits v1.0 fallback in this state.

### Latent improvements (not bugs):
- **Language token case normalization** — `LANG_EXTENSIONS` keys are lowercase, but `services.language` is agent-emitted. A defensive `language?.toLowerCase()` in `auth-db-extractor.js collectSourceFiles` would harden against agent output variation.
- **ENR-05 spring.datasource.url** — currently detected via in-`.java`-file comments. A follow-up could add cross-extension scanning for `application.{yml,properties}` config files.

### Pre-existing (out of scope):
- **WRKR-07** — bats test failure confirmed pre-existing before Phase 95. Tracked in `.planning/phases/95-shell-cleanup-dispatcher/deferred-items.md`.
