## Integration Check

14/14 cross-phase exports correctly wired after fixes.

### Issues Found and Resolved

1. **`crossing` field dropped in `writeScan`** (Critical)
   - `database.js` called `upsertConnection` without `conn.crossing`
   - Entire actor pipeline was a no-op on live scans
   - Fixed: Added `crossing: conn.crossing || null` to the call
   - Affected: ACTOR-01 → ACTOR-04, INTEL-01 → INTEL-03

2. **`node.boundary` never set on nodes** (Functional)
   - `graph.js` passed boundaries to layout but never annotated nodes
   - Boundary filter dropdown permanently empty
   - Fixed: Built reverse map from `raw.boundaries` → set `boundary` on each node
   - Affected: CTRL-06

### Forward-Looking Schema

`node_metadata` table (DATA-03) exists but has no readers/writers in v3.0. This is intentional — it's the extension point for future STRIDE, vulnerability, and deployment views.
