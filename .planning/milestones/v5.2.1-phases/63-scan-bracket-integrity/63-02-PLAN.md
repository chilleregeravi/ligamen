<objective>
Add a NULL scan_version_id garbage collection step inside endScan() so that legacy rows written
before the version bracket was introduced are deleted after the first successful bracketed scan.

Purpose: Pre-bracket scans left every services/connections row with scan_version_id=NULL. The
existing _stmtDeleteStaleServices / _stmtDeleteStaleConnections filter on IS NOT NULL so
those rows survive indefinitely. After endScan marks the new bracket complete, it must also
remove any remaining NULL rows for the same repo — they are definitively stale.

Output: endScan() with two additional DELETE statements targeting NULL scan_version_id rows,
executed after the standard stale-row cleanup.
</objective>
