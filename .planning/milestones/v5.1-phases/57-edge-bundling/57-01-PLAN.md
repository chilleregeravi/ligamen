    2. Add `computeEdgeBundles` to the import from `./utils.js`.
    3. At the start of `render()`, after `visibleIds` and before the Draw edges section, compute bundles fresh each frame:
       ```js
       // Filter edges that pass active protocol and mismatch filters, then bundle
       const filteredEdges = state.graphData.edges.filter(edge => {
         if (!state.activeProtocols.has(edge.protocol)) return false;
         if (state.mismatchesOnly && !edge.mismatch) return false;
         if (!visibleIds.has(edge.source_service_id) && !visibleIds.has(edge.target_service_id)) return false;
         return true;
       });
       const bundles = computeEdgeBundles(filteredEdges);
       ```
    4. Replace the entire `// Draw edges` for-loop (lines ~150-244 in current file) with a new loop over `bundles`. For each bundle:

       **Determine color:**
       - If bundle has a selected/blast relationship: same logic as before but applied to `bundle.source_service_id` / `bundle.target_service_id`
       - If hasMismatch: color = "#fc8181"
       - Else: color = PROTOCOL_COLORS[bundle.protocol] || COLORS.edge.default

       **Determine lineWidth:**
       - For single edges (count === 1): `1 / state.transform.scale` (unchanged)
       - For bundles (count > 1): `(2 + Math.min(bundle.count - 1, 4)) / state.transform.scale`
         (count=2 → 3px, count=3 → 4px, count=5+ → 6px)
       - Selected or blast bundle: add 1px on top of the above

       **Line dash:** Use PROTOCOL_LINE_DASH[bundle.protocol] for single edges. For bundles (count > 1): use solid line `[]` regardless of protocol — the thickness communicates the bundle.

       **Draw the line and arrowhead** — same pattern as before but using bundle.source_service_id and bundle.target_service_id.

       **Mismatch cross:** Draw if bundle.hasMismatch (same logic as before — red cross at midpoint).

       **Count badge:** Draw if bundle.count > 1:
       - Position: midpoint of the line (mx, my)
       - Offset the badge slightly off the line so it does not overlap the mismatch cross: shift 12px perpendicular to the edge direction
       - Badge background: filled circle radius = `10 / state.transform.scale`, fillStyle = "#1a202c", globalAlpha = 0.9
       - Badge text: `String(bundle.count)`, font = `bold ${Math.round(11 / state.transform.scale)}px system-ui, sans-serif`, fillStyle = "#e2e8f0", textAlign = "center", textBaseline = "middle"
       - Draw background circle first, then text on top

    5. Remove the old per-edge iteration — it is fully replaced by the bundle loop.

    NOTE: Do NOT call `computeEdgeBundles` on `state.edgeBundles` (the state cache). Compute fresh from `filteredEdges` inside `render()` — this keeps the render function self-contained and avoids stale cache issues.
  </action>
